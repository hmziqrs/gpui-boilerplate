gpui-starter gives you a working desktop app in minutes, not days. Instead of wiring up navigation, themes, i18n, and form handling from scratch, you start with everything already working.

## What's included

- **Multi-page architecture** with sidebar navigation
- **21 built-in themes** with live switching
- **i18n** for English and Chinese
- **Form validation** with derive macros
- **Cmd+K command launcher**
- **macOS system tray** integration
- **Secure credential storage** via OS keychain
- **SQLite** for local data

The idea is simple: **a boilerplate, not a framework**. All the code is yours to modify. There's no magic layer between you and GPUI.
